---

# Mission

FellowMart exists to help local businesses establish an online presence without paying recurring SaaS fees or hiring expensive developers.

Many small stores still rely on WhatsApp, Excel sheets, notebooks, and phone calls to manage their business. FellowMart provides a modern, easy-to-use platform where they can manage products, inventory, customers, and orders—all from a single dashboard.

---

# Vision

To become the go-to open-source commerce platform for local retailers, enabling anyone to launch and manage an online store using only free and open-source technologies.

# Tech Philosophy

FellowMart is intentionally built using technologies that anyone can learn, contribute to, and deploy without recurring infrastructure costs.

## Stack

- Next.js
- TypeScript
- Supabase
- PostgreSQL
- Supabase Auth
- Supabase Storage
- Tailwind CSS
- shadcn/ui
- Cloudflare Pages
- Cloudflare Workers (future)
- Open-source libraries only

No proprietary AI services.

No paid backend services.

No vendor lock-in.

---

# MVP Goal

A local business owner should be able to:

1. Create an account.
2. Create a store.
3. Upload products.
4. Share their storefront.
5. Receive customer orders.
6. Manage inventory.
7. Track order status.

...all within **15 minutes**.

---

# Long-Term Vision

FellowMart is not trying to become another Shopify clone.

Instead, it aims to become a lightweight, open-source commerce platform tailored for local retailers, small businesses, and community stores—especially in regions where subscription costs and technical complexity are major barriers to digital adoption.
